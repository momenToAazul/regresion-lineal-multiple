A diferencia de la regresin simple, este enfoque resulta ms apropiado para problemas reales, ya que integra la influencia conjunta de diferentes factores y ofrece una visin ms completa del fenmeno estudiado.

El anlisis se desarrolla con una base de datos de la NASA, originalmente publicada en el NASA Reference Publication 1218 y disponible en el UCI Machine Learning Repository. Este conjunto comprende 1,052 observaciones con seis mediciones clave del comportamiento aerodinmico: frecuencia, ngulo de ataque, longitud de cuerda, velocidad de flujo libre, espesor del desplazamiento y presin sonora. La variable objetivo a modelar es la presin sonora, considerada un indicador del rendimiento aerodinmico.
